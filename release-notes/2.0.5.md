# Release Notes — 2.0.5

## Persistent Tags

- Custom tags now persist permanently and no longer disappear when switching tabs or refreshing the page
- Tag suggestions are sorted alphabetically for easier browsing
- Removed the 12-tag suggestion limit — all tags are now always visible as suggestion chips
- Custom tags sync to the cloud across devices

## Manage Tags

- Added a Manage Tags button inside the Tags dialog that opens a dedicated view for renaming or deleting tags
- Renaming a tag updates it across all rows on every tab in one step
- Deleting a tag removes it from all rows and from the saved suggestions list

## Security

- Performed a security audit and applied hardening improvements across the app
