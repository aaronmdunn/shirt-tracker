Help

Header Buttons
Add Column — Add a new column to the current tab. You can choose from text, dropdown, number, notes, photo, or tags.
Edit Columns — Rename, reorder, or remove columns. Drag columns to rearrange them.
Share Columns — Create a public read-only link so others can view your inventory. You can choose which columns are visible.
Copy Share Link — Copy your public share link to the clipboard.
Cloud Sync — Manually save your data and sync it to the cloud. Your data also auto-saves periodically.
Table Controls
Add Row — Add a new blank row to the table.
Go to Bottom / Go to Top — Jump to the bottom or top of the table.
Filter — Filter rows by typing in the filter bar. You can filter by a specific column or across all columns.
Reset Filters — Clear all active filters and show all rows.
Tabs
Add — Create a new tab to organize your inventory into separate collections.
Delete / Edit — Delete or rename the current tab.
Column Definitions
Condition Abbreviations:
NWT — New With Tags
NWOT — New Without Tags
EUC — Excellent Used Condition
Other — Other
Size Options: XS, S, M, L, XL, 2X, 3X, 4X, 5X
Selection Actions
Delete Item(s) — Delete all rows that are selected with checkboxes.
Tag Item(s) — Add or remove tags on all selected rows at once.
Cloud Backup
Verify Cloud Status — Check if your local data matches what’s saved in the cloud.
Edit Profile Name — Change your display name.
Export CSV — Download the current tab’s data as a CSV spreadsheet.
Import CSV — Upload a CSV file to add rows to the current tab.
Danger Zone
Reset Tab — Delete all rows on the current tab. This can be undone with the Undo button.
Wipe & Reset — Erase everything and start fresh with default columns. This cannot be undone.
Recycle Bin — View and restore recently deleted items. Items are kept for 30 days before being permanently removed.
Other
Icon cells — Tap the icon to the left of each row to pick or change the icon for that shirt type.
Dropdowns — All dropdown columns have an “Add new...” option at the bottom to create new values on the fly.
Photos — Tap a photo thumbnail to view, upload, or clear the photo for that row.
